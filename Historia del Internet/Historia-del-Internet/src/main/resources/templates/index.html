Etiquetas Principales en HTML
Etiqueta Descripcion Ejemplo
<html> Define el documento HTML <html></html>
<head> Contiene metodos/informacion sobre el comumentos <head></head>
<title> Defien el titulo del documento <head> Titulo </head>
<meta> Define metadatos sobre el documento <meta charset="UFC-8">
<body> Define el cuerpo del documento <body></body>
<h1> a <h6> Define encabezados HTML <h1> Encabezado </h1>
<p> Define un parrafo <h1> Hola Mundo</h1>
<a> Crear enlaces <a href="https//www.google.es">Google</a>
<img> Insertar imagnes <img="imagen.jpg" alt="Ejemplo">
<table> Define una tabla <table></table>
<tr> Define una fila en una tabla <tr></tr>
<th> Define una celda de encabezado en una tabla <th>Encabezado</th>
<td> Define una celda en una tabla <td>Celda</td>
<thead> Define el encabezado de una tabla <thead></thead>
<tbody> Define el cuerpo de una tabla <tbody></tbody>
<tfoot> Define el pie de una tabla <tfoot></tfoot>
<caption> Define un título para la tabla <caption>Título</caption>
<form> Define un formulario HTML <form></form>
<input> Define un control de entrada <input type="text">
<button> Define un botón <button>Botón</button>
<div> Define una división o sección <div>Contenido</div>
<span> Define una sección en línea <span>Texto</span>
<br> Inserta un salto de línea <br>
<hr> Define una línea horizontal <hr>
